Markdown学习笔记

发布于 2020-07-10 | 分类于 Markdown | 6分钟 | 1375字数

使用gridea简单搭建了个人博客，想要高效利用，当然是从学会Markdown开始啦。

这篇博客总结如何使用Typora和Gridea编辑Markdown博客。

编辑器：Gridea、Typora

参考网站：菜鸟教程

编辑器使用

Gridea

Gridea主要用来管理博客文章和发布，可设定文章分类，URL等文章信息，支持Markdown编辑（源代码格式，可预览），但功能较少，使用起来不太方便，对新手最友好的办法是Gridea新建博客，设置好博客参数，如图：
!
如果用Gridea源代码打开设置好的md文件则会发现，这些设置都保存在md文件的开头，并以分隔符分隔，如下：

---
title: 'Markdown学习笔记'
date: 2020-07-10 16:25:47
tags: [Markdown]
published: true
hideInList: false
feature: 
isTop: false
---

注意：Gridea只能识别目录下带参数设置头的md文件，推荐在面板设置好再转用Typora编辑，或熟练后直接编辑即可。

Typora

比较流行的Markdown编辑器，编辑模式和功能多样，对新手友好，若不熟悉Markdown源码也可流畅使用，界面多样，编写体验满分。

Markdown语法

标题

方法一：
在标题下方一行标记多个 =（一级标题）或 -（二级标题）：

一级标题
=======
二级标题
-------

方法二（推荐）：

使用 # 表示1-6级标题，几级对应几个 # ，中间用空格隔开：

# 一级标题
## 二级标题

段落

一段话间的行间距较小，使用Enter换行可另起一段，段落之间的间距为一行。

字体

文本两边使用*或者_为斜体，文本两边使用**或者__为粗体。

文本两边使用 ***或者___为粗斜体。

*斜体*
**粗体**
***粗斜体***
_斜体_
__粗体__
___粗斜体___

分隔线

一行中使用三个以上的*、-、_表示分割线，中间可有空格。

**** *
--- --
___ __

删除线

文本两端使用~~，文本将被划掉。

~~删除线~~

下划线

使用HTML<u>标签实现。

<u>下划线</u>

脚注

效果：[1]

[^脚注名]
对应的必须有说明，可放在文章末尾。
[^脚注名]:脚注的说明

列表

无序

使用一个*、+、-做为标记，空格后加上列表名。

* 第一项
+ 第二项
- 第三项

有序

使用数字加上.做为标记，空格后加上列表名。

1. 第一项
2. 第二项
3. 第三项

列表可嵌套，子列表需要四个空格的缩进。

1. 第一项
    * 第一小项
    * 第二小项
2. 第二项

区块

每行的开头使用>+空格，可建立一个区块，区块和列表可以互相嵌套。

> 第一层区块
>> 第二层区块
>> 1. 第一项
>> 2. 第二项
>>> 第三层区块
>>> + 第一项
>>> + 第二项

+ 第一项
    > 使用四个空格
    >
+ 第二项

第一层区块

第二层区块

  1. 第一项
  2. 第二项

第三层区块

  • 第一项
  • 第二项
  • 第一项

    使用四个空格
    使用四个空格

  • 第二项

代码块

段落中的代码文本两边使用反引号`。

`print()`函数

print()函数

代码区块使用四个空格或TAB键。

	import requests
	def get_url(url):
		response=requests.get(url,headers={'User-Agent':'mazzia'},timeout=None)

或者用```包裹一整段代码，开头可指定语言。

​```python
import requests
def get_url(url):
	response=requests.get(url,headers={'User-Agent':'mazzia'},timeout=None)
​```

链接

[链接名称](链接地址)  或  <链接地址>
<https://baidu.com>
[百度](https://baidu.com)
高级链接：
[baidu][1]
必须对应：
[1]:https://baidu.com

图片

![alt 属性文本](图片地址)
![alt 属性文本](图片地址 "可选标题")
或高级链接：
[picture][1]
必须对应：
[1]:图片地址

还可以使用更多功能的<img>标签
